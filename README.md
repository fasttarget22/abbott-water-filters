# Abbott Water Filters — Business Management System

Complete water filter shop management with role-based dashboards.

- **Manager:** Full control — customers, expiry alerts, WhatsApp, feedback, schedule, inventory, employees
- **Technician:** My jobs, parts log, job locations
- **Sales Staff:** Reminders, daily sales, customer view

**Demo:** EMP-001 / 1234 (Manager) | EMP-002 / 2222 (Technician) | EMP-004 / 4444 (Sales)

Built for Abbott Water Filters, Abbottabad
